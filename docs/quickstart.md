# MatsuriOps クイックスタートガイド

> 5分で始められる基本操作ガイド

---

## はじめに

### MatsuriOpsとは

MatsuriOpsは、地域の祭りやイベントの運営を効率化するWebアプリケーションです。

**主な機能:**
- 祭り情報の一元管理
- タスク・スケジュール管理
- 予算・経費管理
- スタッフ配置・シフト管理
- 当日の運営支援（インシデント対応、エリア監視）
- リアルタイムコミュニケーション

### 動作環境

| 環境 | 対応状況 |
|------|---------|
| Chrome（推奨） | ✅ |
| Safari | ✅ |
| Firefox | ✅ |
| Edge | ✅ |
| スマートフォン | ✅ PWA対応 |

## Step 1: アカウント登録（2分）

### 1.1 登録画面へアクセス

ブラウザでMatsuriOpsのURLを開き、「新規登録」をクリックします。

![登録画面](images/ss_register.png)

### 1.2 メールアドレスを入力

1. メールアドレスを入力
2. 「登録」ボタンをクリック

### 1.3 確認メールからログイン

1. 入力したメールアドレスに確認メールが届きます
2. メール内のリンクをクリックしてログイン完了

> **ヒント:** メールが届かない場合は、迷惑メールフォルダを確認してください。

---

## Step 2: プロフィール設定（1分）

### 2.1 設定画面へ移動

ログイン後、右上のメニューから「設定」を選択します。

![設定画面](images/ss_settings.png)

### 2.2 基本情報を入力

以下の情報を入力して「保存」をクリック：

| 項目 | 説明 |
|------|------|
| 名前 | 表示名（他のユーザーに見える名前） |
| 電話番号 | 緊急連絡用 |
| 所属 | 組織・団体名 |
| 緊急連絡先 | 緊急時の連絡先 |
| スキル | 担当可能な作業（設営、運営など） |

---

## Step 3: 祭りに参加/作成（2分）

### 3.1 祭り一覧を確認

ログイン後、祭り一覧ページが表示されます。

![祭り一覧](images/ss_festival_list.png)

### 3.2 既存の祭りに参加する場合

1. 一覧から参加したい祭りをクリック
2. 管理者から招待を受けている場合、メンバーとして追加されます

### 3.3 新しい祭りを作成する場合

1. 「新規作成」ボタンをクリック
2. 祭り情報を入力：
   - 祭り名（必須）
   - 開催日（開始日・終了日）
   - 会場名・住所
   - 規模（小規模/中規模/大規模）
   - 予想来場者数・出店数

![祭り作成](images/ss_festival_form.png)

3. 「作成」をクリックして完了

## トラブルシューティング

### ログインできない

1. メールアドレスが正しいか確認
2. 迷惑メールフォルダを確認
3. 再度「ログイン」から手続きをやり直す

### 画面が正しく表示されない

1. ブラウザを最新版に更新
2. キャッシュをクリア（Ctrl+Shift+R または Cmd+Shift+R）
3. 別のブラウザで試す

### その他の問題

管理者またはサポートに連絡してください。

## PWAとしてインストール

スマートフォンでMatsuriOpsをアプリのように使うことができます。

### iPhoneの場合
1. Safariでアクセス
2. 共有ボタン（□↑）をタップ
3. 「ホーム画面に追加」を選択

### Androidの場合
1. Chromeでアクセス
2. メニュー（⋮）をタップ
3. 「ホーム画面に追加」を選択
