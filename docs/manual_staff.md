## 1. ログインと基本操作

### 1.1 ログイン方法

1. ブラウザでMatsuriOpsにアクセス
2. 「ログイン」をクリック
3. メールアドレスを入力
4. 届いたメールのリンクをクリック

![ログイン画面](images/ss_login.png)

### 1.2 ホーム画面

ログイン後、参加している祭りの一覧が表示されます。

![祭り一覧](images/ss_festival_list.png)

### 1.3 祭り詳細画面

祭りを選択すると、詳細画面が表示されます。

![祭り詳細](images/ss_festival_show.png)

ここから各機能にアクセスできます：
- タスク管理
- シフト管理
- チャット
- お知らせ
- ドキュメント

## 2. 自分のタスク確認

### 2.1 タスク一覧を開く

祭り詳細画面から「タスク管理」をクリックします。

![タスク一覧](images/ss_task_list.png)

### 2.2 タスクの見方

| 表示 | 意味 |
|------|------|
| 🔴 未着手 | まだ開始していないタスク |
| 🔵 進行中 | 現在作業中のタスク |
| 🟢 完了 | 完了したタスク |
| 🟡 保留 | 一時停止中のタスク |
| ⚪ 中止 | キャンセルされたタスク |

**優先度:**
| 表示 | 意味 |
|------|------|
| 🔴 緊急 | 最優先で対応 |
| 🟠 高 | 優先的に対応 |
| 🟡 中 | 通常の優先度 |
| 🟢 低 | 余裕があれば対応 |

### 2.3 タスクの詳細を確認

タスクをクリックすると詳細が表示されます：
- タスクの説明
- 期限
- チェックリスト
- サブタスク

### 2.4 タスクのステータス更新

**リーダーの場合:**
1. タスクのステータスをクリック
2. 新しいステータスを選択
3. 自動的に保存されます

**スタッフ・ボランティアの場合:**
ステータスの更新には管理者への報告が必要です。

### 2.5 チェックリストの操作

タスク詳細画面で：
1. チェックリスト項目をクリック
2. 完了/未完了が切り替わります

## 3. シフト確認

### 3.1 シフト一覧を開く

祭り詳細画面から「シフト管理」をクリックします。

![シフト一覧](images/ss_shift_list.png)

### 3.2 シフトの見方

シフトは日付ごとにまとめて表示されます：
- シフト名
- 開始時刻 〜 終了時刻
- 担当場所
- 必要人数

### 3.3 自分のシフトを確認

自分が担当するシフトは、自分の名前が表示されています。
当日は、開始時刻までに担当場所に集合してください。

## 4. 当日の操作

### 4.1 運営ダッシュボード

祭り詳細画面から「運営ダッシュボード」をクリックします。

![運営ダッシュボード](images/ss_operations.png)

ここでは：
- 現在のインシデント状況
- エリアの混雑状況
- 天候情報

が確認できます。

### 4.2 インシデント報告

問題が発生した場合は、すぐに報告しましょう。

1. 「インシデント報告」ボタンをクリック
2. 以下の情報を入力：

![インシデント報告](images/ss_incident_form.png)

| 項目 | 入力内容 |
|------|---------|
| タイトル | 何が起きたか簡潔に |
| 重大度 | 緊急度を選択 |
| カテゴリ | 種類を選択 |
| 場所 | 発生場所 |
| 説明 | 詳細な状況 |

3. 「報告」をクリック

**重大度の目安:**
| 重大度 | 状況の例 |
|--------|---------|
| 緊急 | 救急対応が必要、火災、大規模な事故 |
| 高 | けが人発生、設備の重大な故障 |
| 中 | 軽い怪我、設備の軽微な問題 |
| 低 | 遺失物、軽微な問い合わせ |

### 4.3 位置情報の共有

チームメンバーに自分の位置を知らせることができます。

1. 祭り詳細から「位置情報」をクリック
2. 「現在位置を共有」をクリック
3. 位置情報の許可を求められたら「許可」を選択

> **注意:** バッテリー消費が増える場合があります。

### 4.4 エリア状況の確認

運営ダッシュボードでエリアの状況を確認できます：

| 混雑度 | 状況 |
|--------|------|
| 0（空き） | 人がほとんどいない |
| 1（閑散） | 余裕がある |
| 2（やや混雑） | 少し混んできた |
| 3（混雑） | 混雑している |
| 4（非常に混雑） | かなり混雑 |
| 5（過密） | 危険な混雑状態 |

## 5. チャット・連絡

### 5.1 チャットを開く

祭り詳細画面から「チャット」をクリックします。

![チャット](images/ss_chat_room.png)

### 5.2 チャットルームの種類

| ルーム | 用途 |
|--------|------|
| 一般 | 全体への連絡 |
| 緊急 | 緊急時の連絡（重要なメッセージのみ） |
| スタッフ | スタッフ間の連絡 |
| 出店者 | 出店者への連絡 |

### 5.3 メッセージの送信

1. チャットルームを選択
2. メッセージを入力
3. 「送信」をクリック

### 5.4 注意事項

- 緊急ルームは本当に緊急な場合のみ使用
- 個人情報は送信しない
- 不要な雑談は控える

## 6. お知らせ確認

### 6.1 お知らせを開く

祭り詳細画面から「お知らせ」をクリックします。

![お知らせ](images/ss_announcements.png)

### 6.2 お知らせの見方

| 優先度 | 表示 | 内容 |
|--------|------|------|
| 緊急 | 🔴 赤枠 | 至急確認が必要 |
| 高 | 🟠 オレンジ枠 | 重要な連絡 |
| 通常 | ⚪ 通常表示 | 一般的な連絡 |

### 6.3 プッシュ通知

お知らせはプッシュ通知でも届きます。
通知を有効にしておくと、重要な連絡を見逃しません。

## よくある質問

### Q: パスワードを忘れました

A: パスワードは使用していません。メールアドレスでログインすると、確認メールが届きます。

### Q: シフトを変更してほしい

A: リーダーまたは管理者に連絡してください。

### Q: インシデント報告を間違えました

A: インシデントを編集して修正するか、管理者に連絡してください。

### Q: 通知が届きません

A: ブラウザの通知設定を確認してください。また、PWAとしてインストールすると通知を受け取りやすくなります。

### Q: アプリが動かなくなりました

A: ブラウザを再読み込み（F5 または Cmd+R）してください。

問題が発生した場合は、以下に連絡してください：

| 状況 | 連絡先 |
|------|--------|

| 緊急事態 | 緊急チャットルーム |
