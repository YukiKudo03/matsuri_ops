## 前提：認証ヘルパー戦略

Wallabyのサンドボックス制限により、**UIを通じてユーザー登録→ログイン**のフローを
各テストスイートの前提として実行する。

```
共通フロー:
1. /users/register でユーザー登録
2. DB直接操作でconfirmed_at設定 + ロール付与
3. /users/log-in でMagic Linkトークン取得
4. /users/log-in/:token でログイン完了
```
